Lokale Dienste

GitHub Pages versucht lokale IPv4-Ziele direkt mit Browser-Freigabe fuer Local Network Access. Wenn Edge/Chrome die Kachel blockiert oder keine Freigabe zeigt, starte die Wall lokal per HTTP.

Kurztutorial

Login-Websites

Wähle als Typ Login-Website. Dann wird der Frame ohne Sandbox geladen. Formulare, Cookies und Login-Popups haben dadurch die besten Chancen.

Grenze: Wenn die Zielseite selbst X-Frame-Options, frame-ancestors, blockierte Third-Party-Cookies oder eine Weiterleitung aus dem Frame nutzt, kann die Regie Wall das nicht erzwingen. Nutze dann Login/Öffnen, melde dich im normalen Tab an und lade danach den Frame neu.

IPv4-/LAN-Dienste

Loopback-Testserver wie http://127.0.0.1:8000 und LAN-Geräte wie http://192.168.0.50:8080 werden aus GitHub Pages direkt versucht. Aktuelle Chromium-/Edge-Browser verlangen dafuer Local Network Access; wenn der Browser fragt, Zugriff erlauben.

  • Empfohlen: src\modules\regie\start-local-regie-wall.bat starten. Das Script hostet das Repository lokal und oeffnet die Wall auf Port 8765.
  • Fallback: Wenn der Browser keine Local-Network-Freigabe gibt, erst das Script starten, dann Lokale Regie mit Setup oeffnen klicken. Die aktuelle Quellenliste wird per URL uebernommen.
  • Grenze: Wenn die Zielseite selbst X-Frame-Options oder frame-ancestors setzt, muss sie im eigenen Tab laufen; das kann keine Wall ueberschreiben.

Tonsteuerung

Mute/Solo/Volume funktionieren bei YouTube und HLS. Bei fremden Website-iFrames ist Tonsteuerung aus Sicherheitsgründen nicht zuverlässig möglich.

Webcam-Suche

Gib einen Ort ein. Die Wall sortiert nur direkt einbettbare Livestreams nach Entfernung. Die WorldCam-Karte dient als externe Recherche, weil viele Webcam-Seiten iFrames blockieren.

Sender suchen

Im Livestream-Katalog oder im Source-Menü kannst du Sendernamen wie BLK TV, Burgenlandkreis, ARD, ZDF, DW, Sky, Al Jazeera, MDR Thüringen, WDR oder ARTE suchen. Beim Übernehmen wird der hinterlegte HLS-, YouTube- oder Website-Link geladen.
