# Theoretische Informatik – Trainer (GitHub Pages)

## Deploy
1. Repo erstellen, Dateien committen
2. Settings → Pages → Deploy from branch
3. Branch `main`, Folder `/ (root)`

## Inhalte
- `automaten.html` – Lehrbuchseite + Struktur
- `rote-boxen.html` – Vollständige Sammlung der „roten Boxen“ (Wortlaut + LaTeX)
- `personen.html` – Timeline + eingebettete von-Neumann-Visualisierung (`vna/index.html`)
