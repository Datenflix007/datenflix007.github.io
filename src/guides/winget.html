WinGet Guide

Der Windows Package Manager – Software schnell über die PowerShell installieren, aktualisieren und verwalten.

Der Windows Package Manager (WinGet) ist ein Kommandozeilen-Tool von Microsoft, mit dem du Software schnell über die PowerShell installieren, aktualisieren und verwalten kannst – ähnlich wie apt, brew oder npm.

Voraussetzungen

WinGet ist auf Windows 10 (ab Version 1809) und Windows 11 meist bereits vorinstalliert.

Prüfen, ob WinGet installiert ist

winget --version

Wenn eine Versionsnummer angezeigt wird, ist alles bereit. Falls nicht, installiere den App Installer aus dem Microsoft Store.

Programme suchen

winget search vscode
  • Suche nach Namen: winget search chrome
  • Suche nach Publisher: winget search --source winget Microsoft

Programme installieren

winget install Microsoft.VisualStudioCode
winget install --id Microsoft.VisualStudioCode -e

Nützliche Optionen:

OptionBedeutung
-eExakte Übereinstimmung der ID
--silentInstallation ohne Benutzeroberfläche
--accept-package-agreementsLizenz automatisch akzeptieren
--accept-source-agreementsQuellenbedingungen akzeptieren

Stille Installation:

winget install --id Google.Chrome -e --silent --accept-package-agreements --accept-source-agreements

Installierte Programme anzeigen

winget list
winget list vscode

Programme aktualisieren

winget upgrade
winget upgrade Microsoft.VisualStudioCode
winget upgrade --all --silent --accept-package-agreements --accept-source-agreements

Programme deinstallieren

winget uninstall Microsoft.VisualStudioCode
winget uninstall --id Microsoft.VisualStudioCode -e

Paketdetails anzeigen

winget show Microsoft.VisualStudioCode

Mehrere Programme auf einmal installieren (packages.json)

Eine JSON-Datei anlegen:

{
  "$schema": "https://aka.ms/winget-packages.schema.2.0.json",
  "CreationDate": "2024-01-01T00:00:00.000Z",
  "Sources": [
    {
      "Packages": [
        { "PackageIdentifier": "Google.Chrome" },
        { "PackageIdentifier": "Microsoft.VisualStudioCode" },
        { "PackageIdentifier": "7zip.7zip" }
      ],
      "SourceDetails": {
        "Argument": "https://cdn.winget.microsoft.com/cache",
        "Identifier": "Microsoft.Winget.Source_8wekyb3d8bbwe",
        "Name": "winget",
        "Type": "Microsoft.PreIndexed.Package"
      }
    }
  ]
}

Und dann importieren:

winget import -i packages.json

Quellen verwalten

winget source list
winget source update

Einstellungen anpassen

winget settings

Power-User Befehle

# Alle Apps im Hintergrund updaten
winget upgrade --all --silent --accept-package-agreements --accept-source-agreements

# Bestimmte Version installieren
winget install --id Python.Python.3.11 --version 3.11.6

# Portable Version (user scope)
winget install --id Neovim.Neovim --scope user

Hilfe

winget --help
winget install --help
winget upgrade --help

Dienste & Entwicklungsumgebungen installieren

MongoDB Community Server

winget install --id MongoDB.Server -e --accept-package-agreements --accept-source-agreements
Get-Service MongoDB
mongosh

MariaDB Server

winget install --id MariaDB.Server -e --accept-package-agreements --accept-source-agreements
Get-Service MariaDB
mysql -u root -p

PostgreSQL

winget install --id PostgreSQL.PostgreSQL -e --accept-package-agreements --accept-source-agreements
Get-Service postgresql*
psql -U postgres

Python (mehrere Versionen)

winget search Python.Python
winget install --id Python.Python.3.12 -e
winget install --id Python.Python.3.11 -e
py -0        # installierte Versionen prüfen
py -3.11     # bestimmte Version starten

Node.js (LTS)

winget install --id OpenJS.NodeJS.LTS -e
node -v
npm -v

OpenJDK – Eclipse Temurin 17

winget install --id EclipseAdoptium.Temurin.17.JDK -e
java -version
javac -version
